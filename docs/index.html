Interface de Programação para o DESSEM¶

Versão: 0.0.2

Data: 01/02/2023

O idessem é um pacote Python para manipulação dos arquivos de entrada e saída do programa DESSEM. O DESSEM é desenvolvido pelo CEPEL e utilizado para a programação da operação do Sistema Interligado Nacional (SIN).

O idessem oferece:

  • Meios para leitura dos arquivos de entrada e saída do DESSEM

  • Armazenamento e processamento de dados otimizados com o uso de numpy

  • Facilidades para estudo e análise dos dados utilizando DataFrames de Pandas

  • Dados estruturados em modelos com o uso do paradigma de orientação a objetos (OOP)

  • Utilidades de escritas dos arquivos de entrada do DESSEM para elaboração automatizada de estudos
