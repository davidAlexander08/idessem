Relatório com função de produção (AVL_FPHA1.DAT)¶

O relatório com as informações referente aos coeficientes da função de produção é armazenado na classe:

class AvlFpha1(data=<cfinterface.data.blockdata.BlockData object>)[código fonte]¶

Armazena os dados referentes aos coeficientes da função de produção das usinas hidráulicas.

Essa classe lida com as informações de saída fornecidas pelo arquivo AVL_FPHA1.

property tabela¶

Obtém a tabela com informações referente aos coeficientes da função de produção.

  • indiceUsina (int)

  • nomeUsina (str)

  • segmentoFpha (int)

  • fatorCorrecao (float)

  • rhs (float)

  • coeficienteVolumeUtil (float)

  • coeficienteVazaoTurbinada (float)

  • coeficienteVazaoLateral (float)

Retorna:

A tabela como um dataframe

Tipo de retorno:

pd.DataFrame | None
